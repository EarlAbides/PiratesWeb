Design Notes

  • Build-sheet-inspired rows with set-colored gradient backgrounds
  • Point cost badge is the loudest element (large, boxed, left-most)
  • Nationality flags as compact inline badges
  • Cannon pips rendered visually with S/L color differentiation
  • Crew cards gracefully degrade — ship-specific stats replaced with type label
  • Sidebar always visible for filter refinement

Design Notes

  • ~35px row height vs ~68px in Direction A — nearly 2x more visible cards
  • Stats in a single inline row with colon notation (M:4 C:6)
  • Ability text heavily truncated — scanning tool, not reading tool
  • Closer to spreadsheet experience users are familiar with
  • Less visual polish — trades "impressive" for "efficient"
